# Ujian Praktikum Pemrograman Web - CRUD System

## Tema yang Dipilih: Data Mahasiswa
Aplikasi web berbasis CRUD (Create, Read, Update, Delete) sederhana untuk mengelola entitas data mahasiswa.

**Atribut Data yang digunakan:**
- NIM
- Nama Lengkap
- Jurusan
- Foto Profil

**Fungsi Setiap File Yang Digunakan**
- index.php: antarmuka visual yang mengambil data dari database untuk update tambah data
- form.php: untuk menampilkan kembali data lama
- script.js: untuk validasi ketika ingin menghapus data dan mencegah formulir kosong
- proses.php: untuk menjalankan perintah dari index php dan form php
- style.css: untuk desain web
